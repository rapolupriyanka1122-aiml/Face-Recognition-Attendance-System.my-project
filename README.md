# Face Recognition Attendance System

## 📖 Overview
This project is a Python-based Face Recognition Attendance System that automatically detects and recognizes faces using OpenCV and records attendance with the current date and time.

## ✨ Features
- Automatic face detection
- Face recognition using OpenCV
- Attendance recording in CSV format
- Student registration with images
- Real-time webcam recognition

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- Pandas
- CSV
- Haar Cascade Classifier

## 📁 Project Structure

```
Attendance_Project/
│── attendance_system.py
│── face_id.py
│── attendance.csv
│── students.csv
│── images/
```

## ▶️ How to Run

1. Install Python.
2. Install the required libraries:
   ```
   pip install opencv-python numpy pandas
   ```
3. Run:
   ```
   python attendance_system.py
   ```
4. Register student faces.
5. Start the attendance system.

## 🎯 Project Objective

To automate attendance management using face recognition technology, reducing manual work and improving accuracy.
